# Guia per contribuir a aquest projecte

*Gràcies per contribuir a aquest projecte!*  
Segueix aquestes guies per assegurar una bona colaboració

## Estil de Codi

- Utilitza noms de *variables clares* i *descriptives*.
- Mantén el codi *net* i *fàcil de llegir*.
- Escriu comentaris quan calgui, però evita comentaris innecessaris.
- Evita duplicar codi, crea funcions reutilitzables quan sigui possible.

## Pull Requests

Quan vulguis fer una Pull request hauras de:
- Crea una branca nova per a cada canvi o funcionalitat.

- Fes els canvis necessaris i crea un commit amb un missatge clar.

- Puja la branca al repositori amb git push.

- Obre una **Pull Request** explicant breument què has canviat i per què.

- Assegura’t que la branca està actualitzada amb *main* abans de fer el merge. 

## Report d'Issues

Si vols reportar un **error** o **suggerir una millora**:

- Afegir un issue a l'apartat de issues, descrivint amb detalls quin es el problema o la millora que vols que s'apliqui, per veure si es pot solucionar. 

- També es pot posar això en el issue ja creat.
